# 龟牌 - Bullshit Card Game

一个使用Godot 4.5引擎开发的卡牌游戏，基于经典的"Bullshit"（也称为Cheat/I Doubt It）游戏规则。

## 游戏规则

### 基本信息
- **玩家**: 4名（1名人类玩家 + 3名AI）
- **牌组**: 标准52张扑克牌
- **目标**: 最先清空手牌的玩家获胜

### 游戏流程

1. **出牌阶段**
   - 游戏按顺序循环目标等级：Ace → 2 → 3 → ... → King → Ace
   - 轮到你时，选择1-4张手牌，点击"Play Selected Cards"
   - 你必须声称打出当前目标等级的牌
   - 你可以说谎！打出的牌不必真的是目标等级

2. **质疑阶段**
   - 其他玩家打出牌后，你有3秒时间点击"CALL BULLSHIT!"进行质疑
   - 如果质疑成功（对方说谎）：对方拿走所有弃牌
   - 如果质疑失败（对方说真话）：你拿走所有弃牌
   - 如果无人质疑：游戏继续，目标等级递增

3. **胜利条件**
   - 最先清空手牌的玩家获胜

## 如何运行

### 前置要求
- Godot Engine 4.5 或更高版本

### 运行步骤
1. 使用Godot打开项目文件夹
2. 在Godot编辑器中打开 `Main.tscn`
3. 点击"运行场景"（F6）或"运行项目"（F5）

## 控制说明

- **选择卡牌**: 点击手牌中的卡牌（可选择1-4张）
- **出牌**: 选择卡牌后，点击"Play Selected Cards"按钮
- **质疑**: 在其他玩家出牌后的3秒内，点击"CALL BULLSHIT!"按钮
- **重新开始**: 游戏结束后，点击"Restart Game"

## AI行为

### 出牌策略
- AI持有目标等级的牌时，90%几率打真牌
- 没有目标等级时，选择最小的牌虚张声势

### 质疑策略
- 逻辑判断：如果AI知道对方不可能有那么多牌（例如AI自己已经有3张，对方声称2张），必定质疑
- 概率判断：
  - 声称3+张：30%几率质疑
  - 声称2张：15%几率质疑
  - 声称1张：5%几率质疑
- 风险评估：弃牌堆超过20张时，质疑概率减半

## 技术特性

- **完整的状态机**: 清晰的游戏流程控制
- **事件驱动架构**: 使用Godot信号系统解耦逻辑和UI
- **流畅动画**: Tween动画提供视觉反馈
- **智能AI**: 基于概率和逻辑推理的AI对手
- **类型安全**: 使用GDScript 2.0强类型注解

