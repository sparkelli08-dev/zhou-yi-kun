## 开发环境

### 要求
- Godot Engine 4.5 或更高版本

### 运行项目
1. 下载并安装 [Godot Engine 4.5](https://godotengine.org/download)
2. 使用 Godot 打开 `project.godot` 文件
3. 点击运行按钮开始游戏
